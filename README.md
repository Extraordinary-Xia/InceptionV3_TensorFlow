# inception_v3 #
inception_v3 is an implementation of inception v3 using tensorflow according to our guidline.

## dependencies ##
- TensorFlow (>= 0.8)


## Features ##
- train
- save checkpoint
- real time dataaugumentation

## How to use ##
- create datalist in data directory.
- datalist  

```
<image path>,<label number>  
...
```

- train
```
python trainer.py
```
